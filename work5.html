05 : Clouds Accesorios

Sistema de control de ventas y traslado de inventario entre múltiples sucursales, para tienda ubicada en centro comercial de Rancagua, Chile.

Desarrollado bajo el framework basado en PHP: Laravel. Para la base de datos se utilizó SQL.

Entre las caracteristícas que ofrece la herramienta destacan:

- Control de acceso al sistema mediante cuentas de usuarios con roles específicos.

- Mantenedor de productos permitiendo crear, editar, actualizar o borrar productos

- Traslados de stocks entre sucursales: por medio de una tabla dinámica, es posible trasladar el stock de un determinado producto de una sucursal a otra. Esto genera una disminución de stock en la sucursal origen y un aumento en la sucursal destino. En caso de no existir el producto en sucursal destino, el sistema modifica la tabla dinámicamente permitiendo asignar el precio del producto para posteriormente crearlo.

- Gestión de ventas: el requerimiento para esta herramienta fue que sea posible el ingreso de todas las ventas realizadas en un día. Por lo que, finalizada la jornada de ventas, el usuario debe ingresar todas las ventas realizas durante el día. Para ello, se dispone de una tabla dinámica donde cada fila corresponde a una venta especifíca de productos.

- Permitir la importación masiva de productos mediante archivo con formato .xslx (Excel).

- Permitir la exportación masiva de ventas y traslados mediante archivo con formato .xslx (Excel).
